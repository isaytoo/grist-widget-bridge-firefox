Ajouter cet onglet
⚠ Builder Grist Labs détecté — l'ajouter couvrira tous les widgets servis depuis ce domaine, pas seulement celui-ci.
Ouvrez un onglet sur votre page Grist ou sur votre widget pour pouvoir l'ajouter ici en un clic. Vous pouvez aussi utiliser ⚙ Options pour saisir une URL manuellement.
Ajouter une instance Grist
L'API Grist Cloud publique est https://api.getgrist.com (différente de docs.getgrist.com).
Ajouter un widget
Pour un widget servi par un builder (gristlabs.github.io, glitch.me…), saisissez l'origine du builder, pas l'URL du widget.
Ajouter une API externe (CORS)
Ajoute un header Access-Control-Allow-Origin: * sur les réponses de cette API (ex : Cloudinary, S3, Supabase…).
