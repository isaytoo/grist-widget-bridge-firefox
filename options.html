Grist Widget Bridge
Configuration avancée

Cette extension fonctionne avec n'importe quelle instance Grist et n'importe quel hébergeur de widget. Configurez ici les adresses concernées — l'extension n'agit que sur ces URLs.

🟢 Instances Grist

Les serveurs Grist où les widgets sont chargés. L'extension y débloque le CORS de l'API.

🟠 Widgets autorisés

Les domaines des widgets personnalisés (ex: La Forge). L'extension y corrige localStorage et autorise le pont d'upload vers Grist.

Astuce : vous pouvez ajouter des adresses plus rapidement via l'icône de l'extension dans la barre d'outils — un clic « + Comme instance Grist » ou « + Comme widget » suffit.
